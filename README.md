# 美国犯罪率数据可视化项目

## 项目简介

本项目基于2005年美国各州犯罪率数据，创建了一个高质量的数据可视化系统。通过精心设计的饼图阵列，直观展示了不同州各类犯罪案件的比例分布。

## 数据来源

- **数据集**: `crimeRatesByState2005.csv`
- **年份**: 2005年
- **覆盖范围**: 美国51个州（包括华盛顿特区）
- **犯罪类型**: 谋杀、强奸、抢劫、严重攻击、入室盗窃、盗窃、车辆盗窃

## 可视化特点

### 🎨 设计亮点

- **中央州名标识**: 每个饼图中央显示州名，采用圆润矩形框设计
- **统一配色方案**: 7色专业配色，确保视觉一致性
- **优化布局**: 8列7行紧凑排列，最大化利用画布空间
- **高质量输出**: 300 DPI高清图片，适合打印和展示

### 📊 技术实现

- **编程语言**: Python
- **主要库**: 
  - `pandas` - 数据处理
  - `matplotlib` - 图表绘制
  - `numpy` - 数值计算


### 图表说明

运行图展示了2005年美国各州犯罪率分布情况：

- **51个饼图** - 每个饼图代表一个州的犯罪类型分布
- **7种犯罪类型** - 用不同颜色区分：
  - 🔵 谋杀
  - 🟣 强奸  
  - 🟠 抢劫
  - 🔴 严重攻击
  - 🟢 入室盗窃
  - 🟤 盗窃
  - 🟡 车辆盗窃

## 数据洞察

### 关键发现

- **犯罪率最高**: 华盛顿特区 (District of Columbia)
- **犯罪率最低**: 新罕布什尔州 (New Hampshire)
- **最常见犯罪**: 盗窃类案件在大多数州占比最高
- **地区差异**: 不同州在各类犯罪率上存在显著差异

### 统计摘要

脚本运行后会输出详细的统计报告，包括：
- 各类犯罪的平均值和中位数
- 最高和最低犯罪率的州
- 总体犯罪率排名

## 技术特色

### 布局优化

- **智能间距**: 垂直0.15，水平0.02的精确间距控制
- **饼图大小**: 半径1.3，确保清晰可见
- **字体设计**: 11pt粗体州名，圆润边框突出显示

### 视觉设计

- **专业配色**: 精心挑选的7色配色方案
- **中央标识**: 独特的州名中央显示方式
- **图例设计**: 底部横向排列，充分利用空间
